InSilicoPCR version 0.1

Online In Silico PCR Tool: Generate Amplification Results from Your Nucleotide Sequence and Primers

About

InSilicoPCR is a program for unix-like operating systems.
It is a computational tool used to calculate the theoretical PCR product directly from a provided DNA sequence and a given set of primers.  The latest version is 0.1 (2023/Nov).

Input Format*

Nucleotide Fasta format for the sequence fasta.  example (example.fasta)
Tab delimited text file format for the primer sets.  primers (rpoB_primers.txt)

Usage:

Input Parameters: Upload nucleotide sequence fasta file *maximum size 5Mb
